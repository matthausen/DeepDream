This is an example of Tensorflow's DeepDream algorithm


### Requirements
In order to run the following you need at least Python 3.5 and Tensorflow 2.0

Create a virtual environment with Anaconda: 
`conda create -n deep_dream anaconda python=3.6`

Activate the environment: 
`conda activate deep_dream`

Install tensorflow 2.0 min: 
`conda install -c anaconda tensorflow`

Switch to deep_dream environment from your Anaconda toolbar.
